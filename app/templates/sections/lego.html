Visual Programming

Graduation Project - Student

Visual programming makes coding accessible, especially for beginners, by transforming complex code into simple, intuitive visual blocks. With my web app, users can generate Python and C++ code simply by dragging and dropping Lego-like blocks. This approach allows users to build programs, create drawings, and solve programming problems visually, without the need to write code manually. The app automatically converts these blocks into functional Python or C++ code, making it an interactive and user-friendly way to understand coding logic and syntax.
